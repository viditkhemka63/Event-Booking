# Event-Booking
Event booking web-app using GraphQL, Node, MongoDB and React.js where you can create events and create and cancel bookings.

## Installing
```javascript
git clone https://github.com/viditkhemka63/Event-Booking.git
cd Event-Booking

// For Backend
cd backend
npm install 
npm start

// For Frontend
cd frontend
yarn install 
yarn start

```
